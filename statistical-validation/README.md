# 製程改善統計檢定工具（Statistical Validation）

以純網頁（單一 `index.html`，無外部相依）實作 Excel「資料分析」工具箱的各項假設檢定，
用來**驗證製程改善的有效性**：改善前後的差異，到底是真實效果，還是隨機波動？

直接用瀏覽器開啟 `index.html` 即可使用；資料可從 Excel 欄位直接複製貼上。

## 涵蓋的檢定（對照 Excel 資料分析工具箱）

| Excel 資料分析項目 | 本工具分頁 | 檢定對象 |
|---|---|---|
| F 檢定：兩個常態母體變異數的檢定 | F 檢定 | 兩組**變異數**（穩定度） |
| t 檢定：成對母體平均數差異檢定 | t 檢定：成對 | 同一批對象前後量測的**平均差** |
| t 檢定：兩個母體平均數差的檢定（假設變異數相等） | t 檢定：等變異 | 兩批獨立樣本的**平均值** |
| t 檢定：兩個母體平均數差的檢定（假設變異數不相等） | t 檢定：不等變異 | 同上，變異數不等時（Welch） |
| （Excel 以 `CHISQ.TEST` 函數提供） | 卡方檢定 | 類別**計數**：適合度／獨立性 |
| 單因子變異數分析 | 單因子 ANOVA | 三組以上的**平均值** |
| 雙因子變異數分析：無重複試驗 | 雙因子 ANOVA | 兩因子主效果 |
| 雙因子變異數分析：重複試驗 | 雙因子 ANOVA | 兩因子主效果＋**交互作用** |

## 各檢定備註說明

### 1. F 檢定（兩個常態母體變異數的檢定）

**什麼情形下使用？**
- 驗證改善後製程的「變異（穩定度）」是否縮小——Cpk 的提升往往來自變異縮小。
- 兩獨立樣本 t 檢定前的前置判斷：變異數相等 → 等變異 t；不等 → Welch t。

**如何使用？**
1. 改善前後各收集一組獨立樣本。
2. 統計量 `F = s₁²/s₂²`，自由度 `df₁ = n₁−1`、`df₂ = n₂−1`。
3. H₀：σ₁² = σ₂²。p < α → 變異數顯著不同。

**注意事項**
- 對非常態資料敏感；明顯偏態時建議改用 Levene 檢定。
- Excel 報表只給單尾 p 值；檢驗「是否相等」應看雙尾（單尾 × 2）。
- 製程改善常用單尾（H₁：改善後變異較小），檢定力較高。

### 2. 成對 t 檢定（成對母體平均數差異檢定）

**什麼情形下使用？**
- 同一批對象改善前後**各量一次**（同 10 台設備調機前後、同一批樣品前後量測）。
- 可扣除個體差異，檢定力通常高於獨立兩樣本 t 檢定。

**如何使用？**
1. 兩欄資料逐列配對、筆數相同。
2. 對差值 d 做單樣本 t 檢定：`t = d̄ / (s_d/√n)`，`df = n−1`。
3. H₀：μ_d = 0。p < α → 前後平均有顯著差異。

**注意事項**
- 前提：差值近似常態（n ≥ 30 可放寬）。
- 配對必須真實存在；兩批不同產品應改用獨立兩樣本 t 檢定。
- 單尾方向須在收資料前決定。

### 3. 兩獨立樣本 t 檢定（假設變異數相等）

**什麼情形下使用？**
- 改善前抽一批、改善後另抽一批（非同一對象），且 F 檢定顯示兩組變異數無顯著差異。

**如何使用？**
1. 合併變異數 `s_p² = [(n₁−1)s₁² + (n₂−1)s₂²]/(n₁+n₂−2)`。
2. `t = (x̄₁−x̄₂−Δ₀) / √(s_p²(1/n₁+1/n₂))`，`df = n₁+n₂−2`。
3. H₀：μ₁−μ₂ = Δ₀（通常 Δ₀ = 0）。

**注意事項**
- Δ₀ 可設非 0：驗證「至少改善某幅度」時使用。
- 不確定變異數是否相等時，直接用 Welch 版本較穩健。

### 4. 兩獨立樣本 t 檢定（假設變異數不相等，Welch）

**什麼情形下使用？**
- F 檢定顯示兩組變異數不同（改善常同時改變平均與變異）。
- 兩組樣本數差距大、或不確定變異數是否相等時的安全預設。

**如何使用？**
1. `t = (x̄₁−x̄₂−Δ₀) / √(s₁²/n₁ + s₂²/n₂)`。
2. 自由度用 Welch–Satterthwaite 公式估計（Excel 會四捨五入為整數）。

### 5. 卡方檢定（適合度／獨立性）

**什麼情形下使用？**
- 資料是**類別計數**（缺陷類型件數、良/不良品數、各班別異常次數）。
- 適合度：改善後缺陷類型分布是否改變、是否符合歷史比例。
- 獨立性：缺陷是否與班別/機台有關；**「改善前後 × 良/不良」的 2×2 表即改善前後不良率比較**。
- Excel 資料分析工具箱沒有卡方項目，需用 `CHISQ.TEST` 函數。

**如何使用？**
1. `χ² = Σ(O−E)²/E`；適合度 `df = k−1`，獨立性 `df = (列−1)(行−1)`。
2. 獨立性檢定的期望次數 = 列合計 × 行合計 ÷ 總計（工具自動計算）。
3. 永遠是右尾檢定。p < α → 分布不符期望／兩變數不獨立。

**注意事項**
- 每格期望次數建議 ≥ 5；不足時合併類別或改用 Fisher 精確檢定（工具會警示）。
- 必須輸入「次數」，不能輸入比例或平均值（適合度的期望欄例外，會自動換算）。

### 6. 單因子 ANOVA（單因子變異數分析）

**什麼情形下使用？**
- 比較三組以上平均（三種參數水準、多台機、多家供應商）。
- 不可用多次 t 檢定取代：兩兩比較會使整體型一錯誤率膨脹（3 組約 14%）。

**如何使用？**
1. `F = MS組間/MS組內`；H₀：各組平均全相等。
2. p < α → 至少一組不同；接著做事後比較（Tukey HSD 等）找出是哪幾組。

**注意事項**
- 前提：各組獨立、常態、變異數同質。
- 兩組時與等變異 t 檢定等價（F = t²）。

### 7. 雙因子 ANOVA（無重複／重複試驗）

**什麼情形下使用？**
- 同時評估兩個因子（溫度 × 壓力、操作員 × 機台）。
- 無重複（每格 1 筆）：只能檢定主效果；典型用於隨機集區設計。
- 重複試驗（每格 r ≥ 2 筆、平衡）：可額外檢定**交互作用**，是製程窗口優化的關鍵。

**如何使用？**
1. 以長格式輸入：每行「因子A水準, 因子B水準, 數值」；工具自動判斷有無重複。
2. **判讀順序：先看交互作用。** 顯著時主效果不能單獨解讀，改看「哪個組合最好」。

**注意事項**
- 重複試驗必須平衡（每格次數相同）。
- 無重複試驗隱含「無交互作用」假設；懷疑有交互作用務必安排重複。

## 假設檢定通用流程

1. **設定假設**：H₀ =「無差異／改善無效」，H₁ =「有差異／改善有效」。
2. **選 α**：常用 0.05（容許 5% 誤判「沒差異卻說有」的風險）。
3. **算統計量與 p 值**：p = 在 H₀ 為真下，看到目前或更極端資料的機率。
4. **結論**：p < α → 拒絕 H₀（顯著）；p ≥ α → 證據不足（≠ 證明沒差異）。

> ⚠ 統計顯著 ≠ 實務重要。p 值小只代表差異不是巧合；改善幅度是否值得投資，仍要看平均差、變異縮小量與成本效益。

## 技術說明

- 分布函數（t、F、χ²）以正規化不完全 beta／gamma 函數計算（Lanczos 近似 + 連分數），
  臨界值以二分法反解，精度與 Excel `T.DIST`/`F.DIST`/`CHISQ.DIST` 一致（誤差 < 1e-10）。
- 無任何外部函式庫與網路請求，可離線使用。
