如何選擇正確的檢定？

驗證製程改善是否有效，本質上是問：「改善前後的差異，是真實效果還是隨機波動？」 假設檢定用資料回答這個問題。請先依照資料型態與比較目的選擇檢定：

Q1. 你的資料是「連續數值」（尺寸、良率、時間、強度…）還是「類別計數」（缺陷類型件數、不良/良品數…）？
→ 類別計數：使用 卡方檢定（分布是否改變、兩類別變數是否相關）
→ 連續數值：往 Q2
Q2. 你想比較的是「平均值」還是「變異（穩定度）」？
→ 變異：使用 F 檢定（改善後製程是否更穩定）
→ 平均值：往 Q3
Q3. 要比較幾組？
→ 兩組，且是「同一批對象」改善前後各量一次：成對 t 檢定
→ 兩組，且是「兩批獨立樣本」：先做 F 檢定判斷變異數是否相等 → 相等用 等變異 t 檢定，不相等用 不等變異 t 檢定（Welch）
→ 三組以上（多個參數水準、多台機、多供應商）：單因子 ANOVA
→ 同時考慮兩個因子（如溫度 × 壓力）：雙因子 ANOVA

假設檢定的基本邏輯（各分頁通用）

  1. 設定假設：虛無假設 H₀ 通常是「沒有差異／改善無效」，對立假設 H₁ 是「有差異／改善有效」。
  2. 選定顯著水準 α：常用 0.05，代表容許 5% 的機率「其實沒差異卻誤判有差異」（型一錯誤）。
  3. 計算檢定統計量與 p 值：p 值 = 假設 H₀ 為真時，觀察到目前（或更極端）資料的機率。
  4. 下結論：p < α → 拒絕 H₀，差異「統計上顯著」；p ≥ α → 無法拒絕 H₀，證據不足（不等於證明沒差異）。
⚠ 統計顯著 ≠ 實務重要。p 值很小只代表差異「不是巧合」，改善幅度是否值得投資，仍要看平均差、變異縮小量等實際數字與成本效益。
📖 檢定說明：F 檢定（兩個常態母體變異數的檢定）

什麼情形下使用？

  • 想驗證改善後製程的「變異（穩定度）」是否改變——例如導入自動化後，產品尺寸的散布是否縮小。製程能力（Cpk）的提升往往來自變異縮小，而不只是平均值位移。
  • 做「兩獨立樣本 t 檢定」之前的前置判斷：兩組變異數相等 → 用等變異 t 檢定；不相等 → 用不等變異（Welch）t 檢定。

如何使用？

  1. 改善前、改善後各收集一組獨立樣本（不同批次、不同時間點皆可，但兩組之間不能配對）。
  2. 計算兩組樣本變異數 s₁²、s₂²，統計量為兩者比值：
F = s₁² / s₂²　（自由度 df₁ = n₁−1，df₂ = n₂−1）
  1. 若兩母體變異數真的相等，F 應接近 1；F 遠大於或遠小於 1 都代表變異數可能不同。
  2. 比較 p 值與 α：p < α → 兩組變異數顯著不同。

前提假設與注意事項

  • 兩組樣本互相獨立，且母體近似常態分布。F 檢定對非常態非常敏感，資料明顯偏態時結論可能失真（可改用 Levene 檢定）。
  • Excel 的「F 檢定」報表只給單尾 p 值；若只是想檢驗「變異數是否相等」（不預設方向），應看雙尾 p 值（單尾 × 2）。本工具兩者都提供。
  • 製程改善驗證常用單尾：H₁ 設為「改善後變異數 < 改善前」，方向明確時檢定力較高。

判讀範例

改善前 s² = 0.25、改善後 s² = 0.09，F = 2.78，單尾 p = 0.01 < 0.05 → 改善後變異顯著縮小，製程穩定度提升有統計證據支持。

📖 檢定說明：成對 t 檢定（成對母體平均數差異檢定）

什麼情形下使用？

  • 「同一批對象」在改善前後各量測一次，資料一一配對：同 10 台設備調機前後的節拍時間、同 12 件樣品改善前後的量測值、同一組作業員訓練前後的產出。
  • 優點：把「個體之間的差異」（每台機器本來就快慢不同）從誤差中扣除，只看「改善造成的變化」，靈敏度（檢定力）通常比獨立兩樣本 t 檢定高。

如何使用？

  1. 確認兩欄資料筆數相同且逐列配對（第 1 筆對第 1 筆…）。
  2. 計算每一對的差 d = x₁ − x₂，對差值做單樣本 t 檢定：
t = d̄ / ( s_d / √n )　（自由度 df = n−1）
  1. H₀：平均差 μ_d = 0（改善無效）；H₁：μ_d ≠ 0（雙尾）或 >0 / <0（單尾，方向明確時）。
  2. p < α → 改善前後平均有顯著差異。

前提假設與注意事項

  • 「差值 d」近似常態分布（樣本數 ≥ 30 時中央極限定理可放寬）。
  • 配對必須真實存在——兩批不同的產品不能硬湊成對，那要用獨立兩樣本 t 檢定。
  • 若改善方向明確（例如目標是「縮短」時間），用單尾檢定較有力，但方向必須在收資料前決定，不能看完資料再選。

判讀範例

10 台設備調機後平均節拍縮短 1.8 秒，t = 4.2，雙尾 p = 0.002 < 0.05 → 調機效果顯著。

顯著水準 α
假設的均數差（H₀）
結論依據
📖 檢定說明：兩獨立樣本 t 檢定（假設變異數相等）

什麼情形下使用？

  • 比較「兩批獨立樣本」的平均值：改善前抽一批、改善後另抽一批（不是同一批對象量兩次）；或新舊治具、兩家供應商各生產一批。
  • 且事先用 F 檢定確認兩組變異數沒有顯著差異（Excel 的標準流程即：先跑 F 檢定，再選等變異或不等變異 t 檢定）。

如何使用？

  1. 兩組樣本數可以不同。先以 F 檢定確認變異數相等。
  2. 把兩組變異數「合併」成一個共同估計值 s_p²，計算：
s_p² = [ (n₁−1)s₁² + (n₂−1)s₂² ] / (n₁+n₂−2)
t = (x̄₁ − x̄₂ − Δ₀) / √( s_p² (1/n₁ + 1/n₂) )　（df = n₁+n₂−2）
  1. H₀：μ₁ − μ₂ = Δ₀（通常 Δ₀=0，即兩組平均相等）。
  2. p < α → 兩組平均值顯著不同。

前提假設與注意事項

  • 兩組獨立、母體近似常態、變異數相等。若 F 檢定顯示變異數不等，請改用「不等變異」分頁。
  • 不確定變異數是否相等時，直接用不等變異（Welch）版本較穩健，結論通常不會更差。
  • Δ₀ 可設為非 0：例如想驗證「良率至少提升 2%」，把 Δ₀ 設為 −2（變數1=改善前）做單尾檢定。

執行等變異 t 檢定

顯著水準 α
假設的均數差 Δ₀
結論依據
📖 檢定說明：兩獨立樣本 t 檢定（假設變異數不相等，Welch）

什麼情形下使用？

  • 同樣是比較兩批獨立樣本的平均值，但 F 檢定顯示兩組變異數顯著不同——這在製程改善中很常見：改善不只改變了平均，也讓變異縮小了。
  • 兩組樣本數差距很大、或不確定變異數是否相等時，也建議直接使用本檢定（Welch 檢定即使變異數其實相等，結論也幾乎不受影響，是較安全的預設選擇）。

如何使用？

  1. 統計量不合併變異數，各自用各自的：
t = (x̄₁ − x̄₂ − Δ₀) / √( s₁²/n₁ + s₂²/n₂ )

自由度以 Welch–Satterthwaite 公式估計（通常不是整數，Excel 會四捨五入為整數）：

df = (s₁²/n₁ + s₂²/n₂)² / [ (s₁²/n₁)²/(n₁−1) + (s₂²/n₂)²/(n₂−1) ]
  1. H₀：μ₁ − μ₂ = Δ₀；p < α → 兩組平均值顯著不同。

前提假設與注意事項

  • 兩組獨立、母體近似常態；不要求變異數相等。
  • 本工具同時顯示精確 df 與 Excel 取整後的結果，兩者 p 值差異極小。

判讀範例

改善前平均 92.1%（s=2.4）、改善後平均 95.3%（s=0.9），t = −4.8，單尾 p < 0.001 → 良率提升顯著，且變異同時縮小（可再用 F 檢定佐證穩定度改善）。

顯著水準 α
假設的均數差 Δ₀
結論依據
📖 檢定說明：卡方檢定（適合度／獨立性）

什麼情形下使用？

  • 資料是「類別的計數」而不是連續數值：各缺陷類型的件數、良品/不良品數、各班別的異常次數。
  • 適合度檢定：一組類別計數的分布是否符合預期分布——改善後缺陷類型的組成是否改變？是否仍符合歷史比例？
  • 獨立性檢定：兩個類別變數是否相關——缺陷類型是否與生產線/班別/機台有關？改善前後（一個變數）與良/不良（另一個變數）是否獨立？
    ※「改善前後不良率是否下降」正是 2×2 獨立性檢定的典型應用。
  • Excel 的「資料分析」工具箱沒有卡方項目，需使用 CHISQ.TEST(觀察範圍, 期望範圍) 函數；本工具直接輸出統計量、自由度與 p 值。

如何使用？

  1. 整理觀察次數 O。適合度：再準備期望次數 E（可輸入比例，工具會自動換算）；獨立性：輸入列聯表，期望次數由「列合計 × 行合計 ÷ 總計」自動計算。
  2. 計算統計量：
χ² = Σ (O − E)² / E　（適合度 df = k−1；獨立性 df = (列數−1)(行數−1)）
  1. χ² 越大代表觀察值偏離期望越多；p < α → 分布不符合期望／兩變數不獨立。

前提假設與注意事項

  • 各觀察值互相獨立，資料為「次數」而非比例或平均值。
  • 每格期望次數建議 ≥ 5；不足時應合併類別，或（2×2 表）改用 Fisher 精確檢定。本工具會自動警示。
  • 卡方檢定永遠是右尾檢定（只看偏離是否過大），沒有單尾/雙尾之分。

判讀範例

改善前 500 件中不良 40 件、改善後 500 件中不良 18 件，χ² = 8.9，p = 0.003 < 0.05 → 不良率與改善前後「不獨立」，即改善顯著降低不良率。

執行卡方檢定

檢定類型
顯著水準 α

範例格式：列＝改善前/後（或班別），行＝良品/不良品（或缺陷類型），填入「件數」。

📖 檢定說明：單因子變異數分析（One-Way ANOVA）

什麼情形下使用？

  • 比較「三組以上」的平均值：三種製程參數水準、四台機、多家供應商的產品強度是否有差異。
  • 為什麼不做多次 t 檢定？三組兩兩比較要做 3 次 t 檢定，每次都有 5% 誤判風險，整體型一錯誤率會膨脹到約 14%。ANOVA 用一次檢定控制整體錯誤率。

如何使用？

  1. 每組收集獨立樣本（各組樣本數可以不同）。
  2. 把總變異拆成「組間變異」（因子造成）與「組內變異」（隨機誤差）：
F = MS組間 / MS組內 = ( SSB/(k−1) ) / ( SSW/(N−k) )
  1. H₀：所有組平均相等；H₁：至少一組不同。組間變異遠大於組內變異（F 大）→ 拒絕 H₀。
  2. p < α → 各組平均不全相等。但 ANOVA 不會告訴你是哪幾組不同，顯著後需做事後多重比較（如 Tukey HSD），或至少觀察各組平均與信賴區間。

前提假設與注意事項

  • 各組獨立、母體近似常態、各組變異數同質（可先目視各組標準差，相差懸殊時考慮 Welch ANOVA 或資料轉換）。
  • 兩組時 ANOVA 與等變異 t 檢定完全等價（F = t²）。

判讀範例

三種退火溫度的平均硬度 F = 7.6，p = 0.004 < 0.05 → 溫度水準對硬度有顯著影響，接著看哪個水準最佳並做事後比較確認。

執行單因子 ANOVA

顯著水準 α
📖 檢定說明：雙因子變異數分析（Two-Way ANOVA，無重複／重複試驗）

什麼情形下使用？

  • 想「同時」評估兩個因子對品質特性的影響：溫度 × 壓力、操作員 × 機台、材料 × 治具。一次實驗回答兩個問題，比分開做兩次單因子實驗更有效率。
  • 無重複試驗：每個因子組合只量測 1 次（Excel：雙因子變異數分析－無重複試驗）。只能檢定兩個因子各自的主效果，無法估計交互作用。典型用途：隨機集區設計（把「機台」當集區因子，排除機台差異後檢定處理效果）。
  • 重複試驗：每個組合重複量測 r 次（r ≥ 2，且各格次數相同）（Excel：雙因子變異數分析－重複試驗）。可額外檢定交互作用——例如「高溫只有在高壓下才有效」這種組合效果，是製程窗口（process window）優化的關鍵資訊。

如何使用？

  1. 以「長格式」輸入資料：每行一筆觀測「因子A水準, 因子B水準, 數值」。工具會自動判斷每格次數：每格 1 筆 → 無重複；每格 r 筆（各格相同）→ 重複試驗。
  2. 總變異拆解為：因子 A、因子 B、（交互作用 A×B，僅重複試驗）、誤差；各自算 F 值與 p 值。
  3. 判讀順序（重複試驗）：先看交互作用。交互作用顯著時，主效果不能單獨解讀，應改看「哪個組合最好」；交互作用不顯著，才分別解讀 A、B 的主效果。

前提假設與注意事項

  • 各觀測獨立、誤差近似常態且變異數同質；重複試驗需為平衡設計（每格次數相同）。
  • 無重複試驗其實是假設「沒有交互作用」才成立——若懷疑有交互作用，務必安排重複。

判讀範例

溫度 p = 0.01（顯著）、壓力 p = 0.30（不顯著）、交互作用 p = 0.04（顯著）→ 因交互作用顯著，不能說「壓力沒影響」，應看組合平均：高溫+高壓的組合表現最佳。

每格 1 筆 → 無重複試驗；每格重複 r 筆（各格相同）→ 重複試驗（含交互作用）。
